# Todo App

## Using React, Redux, Redux-saga

A simple todo app built with the latest tech

## How to work in this project

Development

    $ yarn 
    $ yarn start
    
Building for production
    
    $ yarn
    $ yarn build
    
Running test

    $ yarn
    $ yarn test
